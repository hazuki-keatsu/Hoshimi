## 技术选型

Rust + Skia + SDL2 + Lua + 原生资源管理 + FFmpeg

1. Rust 作为引擎核心开发语言，依托内存安全、零 GC 的编程范式解决 GalGame 引擎常见的内存泄漏、崩溃问题，编译后高性能无运行时开销；优秀的跨平台编译能力与 cargo 包管理生态，可实现一套代码多平台编译，兼顾开发效率与引擎稳定性。
2. Skia 作为核心跨平台 2D 渲染引擎，原生支持多层 2D 图片叠加、Alpha 透明混合、高质量 PNG/WEBP 加载、圆角毛玻璃对话框、各类转场 Shader 与屏幕滤镜，渲染细腻度与色彩还原度极佳；多语言文本渲染支持完善，可轻松实现文字描边、渐变、逐字浮现等核心文本效果，轻量化无冗余，完美适配纯 2D 的 GalGame 渲染场景。
3. SDL2 作为底层核心支撑库，为引擎提供跨平台窗口创建与管理、标准化 OpenGL 上下文（供 Skia 无缝接管渲染）、全平台统一的键鼠 / 触摸 / 手柄输入事件处理、稳定的游戏主循环调度，同时原生集成成熟的音频播放能力；生态成熟稳定、跨平台兼容性全覆盖，与现有技术栈衔接零成本，无需额外集成第三方库。
4. Lua（Rust 绑定：mlua）作为上层脚本语言，实现剧情逻辑、立绘切换、对话分支、转场触发等业务逻辑与引擎底层的解耦，语法简洁、嵌入成本低，兼顾开发灵活性与运行效率。
5. 基于 Rust 原生库实现资源管理，提供资源懒加载、纹理池复用、跨平台文件路径适配、资源打包加密能力，解决大量资源加载导致的内存与性能问题，轻量化无外部依赖。
6. FFmpeg（Rust 绑定：ffmpeg-next）+ SDL2 实现视频播放，完美支持 GalGame 的 OP/ED 动画需求，音画同步无卡顿，与现有技术栈无缝整合，满足引擎的完整功能需求。
